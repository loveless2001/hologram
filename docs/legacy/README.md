# Legacy Documentation

This folder contains historical material, earlier design framing, and reports that no longer match the current top-level architecture.

These files are kept for reference only.

Typical reasons a file was moved here:
- it describes the old gravity-first architecture as the main system story
- it presents historical phase or pivot notes as if they were still current
- it documents exploratory reports rather than the current default workflow

Use `docs/README.md` and `docs/architecture_design.md` for the current design.
